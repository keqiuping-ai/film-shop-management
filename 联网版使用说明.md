# 联网版使用说明

## 本机启动

双击：

`启动联网版.command`

启动后浏览器打开：

`http://localhost:4318`

默认账号：

- 邮箱：`admin@filmshop.local`
- 密码：`admin123`

首次登录后请到“设置”修改密码。

## 服务端口配置

已经预留服务端口配置文件：

`server-config.json`

默认内容：

```json
{
  "port": 4318,
  "host": "0.0.0.0",
  "publicUrl": ""
}
```

如果上传到服务器后需要改端口，例如服务器要求用 `3000`，把它改成：

```json
{
  "port": 3000,
  "host": "0.0.0.0",
  "publicUrl": "https://app.yourfilmshop.com"
}
```

也可以用服务器环境变量覆盖端口：

```bash
PORT=3000 node server.js
```

服务器健康检查地址：

`/api/health`

例如：

`http://localhost:4318/api/health`

## 系统升级接口预留

你说的“后面系统升级更新用的端口”，这一版已经预留成升级通道，不是普通访问端口。

相关文件：

- `version.json`：当前系统版本号。
- `server-config.json`：升级通道配置。
- `updates/`：以后放升级机制说明或升级包处理逻辑。
- `migrations/`：以后放数据库升级脚本。

相关接口：

- `/api/system/info`：查看当前版本、构建号、升级通道。
- `/api/system/check-update`：检查是否有新版本。

`server-config.json` 里有：

```json
"update": {
  "channel": "stable",
  "checkUrl": "",
  "allowRemoteUpgrade": false
}
```

以后正式部署云端升级服务时，把 `checkUrl` 改成你的升级服务地址，例如：

```json
"update": {
  "channel": "stable",
  "checkUrl": "https://updates.yourfilmshop.com/film-shop/latest.json",
  "allowRemoteUpgrade": false
}
```

安全建议：第一阶段不要开启自动远程升级。正式做自动升级前，必须有 HTTPS、签名校验、升级前备份和失败回滚。

## 同一 Wi-Fi 内多设备使用

在运行服务器的 Mac 上查看本机 IP：

```bash
ipconfig getifaddr en0
```

假设显示 `192.168.1.25`，其他电脑或 iPad 打开：

`http://192.168.1.25:4318`

这样大家访问的是同一台服务器，所以数据互通。

## iPad 安装成 App

1. 用 Safari 打开系统网址。
2. 点分享按钮。
3. 选择“添加到主屏幕”。

## Mac 桌面快捷方式

系统右上角的“安装到桌面”会下载一个 `.webloc` 快捷方式文件。

把这个文件拖到桌面，以后双击即可打开云端系统。

当前这台 Mac 的桌面已经生成：

`/Users/keqiuping/Desktop/美国贴膜店管理系统.webloc`

## 中文 / English 切换

系统右上角有语言按钮：

- 中文界面会显示 `English`，点击后切换到英文。
- 英文界面会显示 `中文`，点击后切换到中文。

语言选择保存在每台设备本地。中国办公室可以用中文，美国员工可以用英文，大家仍然共用同一套订单、库存和客户数据。

## 数据自动同步

系统已经改成秒级实时同步：一台电脑、iPad 或手机保存订单、库存、零售批发、房租水电等数据后，其他已登录设备会自动收到更新，不需要人工刷新。

如果某台设备网络不稳定，系统还会保留 5 分钟自动同步作为兜底；设备重新联网后会继续同步最新数据。

## 库存报警表

每个库存 SKU 都可以设置“预警库存/最低数量”。当当前库存小于或等于这个最低数量时，系统会自动把该 SKU 放进“库存报警”表，用来提前补货。

使用方式：

1. 打开“库存出入库”。
2. 编辑库存商品，设置“当前库存”和“预警库存”。
3. 打开左侧“库存报警”。
4. 查看当前库存、最低数量、建议补货数量和仓位。
5. 点击“补货入库”可以直接生成入库流水。

## 客资和客服提成

系统左侧有“客资提成”模块，用于记录 Yelp、Google Maps、Meta/Facebook、Instagram、网站、电话等互联网客资来源。

使用方式：

1. 先在“客服提成规则”里新增客服或前台人员。
2. 给每个人设置“到店提成”和“成交提成”。
3. 每条客资录入来源平台、客户、电话、服务项目、负责客服和状态。
4. 状态为“已到店”或“已成交”时，计入到店数。
5. 状态为“已成交”时，计入成交数。
6. 系统自动统计到店率，目标为 30%。
7. 系统自动统计到店成交率，目标为 50%。
8. 有提成权限的人可以查看每个客服的总提成。

前台和文员默认可以录入客资，但看不到客服提成金额。老板、店长、财务可以查看提成。

当前支持三类提成方案：

- Judy / 自然到店 / TOB：包工包料按售价阶梯；只卖膜按膜价格 3%（自然到店、电话询问）或 6%（自己关系）。
- Byran / 线上 TOC：线上引流接待按阶梯提成：1000/2000/3000/4000 美元区间，对应 20/30/50/100 美元。
- 情侣 / TOC 满一万：TOC 引流到店成交金额达到 10000 美元后，按阶梯提成。

客资录入时需要选择“客资类型、客户类型、销售类型”。系统会根据负责客服的提成方案自动计算提成。

## 员工权限设置

老板账号进入“账号权限”后，可以给每个员工单独勾选权限。

老板账号受保护：不能在“账号权限”里被停用、删除或改成其他角色。老板自己的邮箱和密码请在“设置 / 我的账号”里修改。其他员工，包括店长、前台、文员、仓库、师傅、财务，都可以由老板账号修改权限。

录入入口：

1. 用老板账号登录。
2. 左侧打开“账号权限”。
3. 点击“新增前台账号”或“新增文员账号”。
4. 填姓名、邮箱和临时密码。
5. 选择角色后，系统会自动套用默认权限，也可以手动勾选或取消。
6. 保存后，把邮箱和临时密码发给员工，让员工自己登录并修改密码。

已经支持的权限包括：

- 查看施工订单
- 录入施工订单
- 编辑施工订单
- 删除施工订单
- 查看车型定价
- 编辑车型定价
- 查看师傅资料
- 查看施工工费
- 编辑师傅和工费
- 查看库存
- 编辑库存/出入库
- 查看零售批发订单
- 编辑零售批发订单
- 查看客资
- 录入/编辑客资
- 查看客服提成
- 编辑客服提成规则
- 查看报表
- 查看完整财报/成本/利润
- 管理员工账号和权限
- 修改系统设置/检查升级

系统会同时做两层控制：

1. 没有权限的菜单和按钮不显示。
2. 服务端接口也会拒绝没有权限的访问。

例如，前台可以只给“查看施工订单、录入施工订单、编辑施工订单”；文员可以给“施工订单、零售批发、库存查看、运营成本录入”；仓库只给“查看库存、编辑库存/出入库”；财务可以给“查看报表、查看完整财报/成本/利润”。

## 公网正式使用

如果中国办公室和美国门店不在同一个网络，需要把这个系统部署到云服务器，并绑定 HTTPS 域名。

推荐：

- 服务器：美国西海岸、香港或新加坡
- 域名：例如 `app.yourfilmshop.com`
- 必须开启：HTTPS、每日备份、强密码、员工独立账号
